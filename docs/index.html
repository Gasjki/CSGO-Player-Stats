About

CS:GO Stats is a simple PHP script that reads data from Valve and then displays them in a unique and attractive design. Moreover, everything happens within 5 seconds. The script displays data about kills, weapons or maps.

Features

- Details about Steam Account - SteamID, member since, VAC situation, Market Situation etc.

- Stats about kills/weapons/maps - Kills on every maps, with every weapon/rounds played/win-lose situation

 

Requirements

  • PHP 5.x or higher version
  • cURL
  • .htaccess Rewrite Engine

 

Installation

To use this script is so easy. All you have to do is :

1) Go to Steam Developer (http://steamcommunity.com/dev/apikey)

2) Make your own API

3) Copy and paste in settings.php between " ".

4) That's all !
